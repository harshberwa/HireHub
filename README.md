# 🚀 HireHub – MERN Job Portal

HireHub is a full-stack job portal built using the MERN stack (MongoDB, Express, React, Node.js). It supports role-based access for Students, HRs, and Admins with real-world features like job posting, applications, analytics, and notifications.

## 🛠 Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Multer (file upload)

---

## 🔐 Features

### 👤 Authentication
- Register & Login (JWT based)
- Role-based access control
- Auto logout (24h session)

---

### 🎓 Student Features
- Browse jobs (search, filter, pagination)
- Apply to jobs
- Save jobs (wishlist ❤️)
- Track application status (pending/selected/rejected)
- Profile management (bio, skills, resume upload)

---

### 🧑‍💼 HR Features
- Post jobs
- Manage own jobs
- View applicants
- Update application status
- HR dashboard

---

### 🛡 Admin Features
- Dashboard overview
- Manage users & jobs
- HR approval system
- Admin analytics (charts 📊)

---

### 🔔 Notifications
- Real-time style notifications
- Application updates
- HR alerts when student applies

---

### 🎨 UI/UX
- Dark / Light mode 🌙☀️
- Glassmorphism UI
- Responsive design
- Smooth animations

---

## 📊 Analytics
- Total Users, Jobs, Applications
- Monthly growth charts
- Role distribution (Student vs HR)
